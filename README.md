# Word Master

7세 아이가 영어 단어 시험을 재미있게 연습할 수 있는 브라우저 게임입니다. 별도 서버나 빌드 과정 없이 `index.html` 파일만 열면 바로 실행됩니다.

## 실행 방법

1. 이 폴더의 `index.html`을 브라우저로 엽니다.
2. 메인 화면에서 원하는 단어 게임을 선택합니다.
3. 3개 단어 게임 중 2개를 100점 맞히면 보너스 게임이 열립니다.

## 게임 구성

- `뜻 보고 단어 쓰기`: 뜻을 보고 영어 단어를 직접 입력합니다.
- `예문 빈칸 맞추기`: 예문 속 빈칸에 알맞은 단어를 보기에서 고릅니다.
- `듣고 뜻 고르기`: 영어 단어 발음을 듣고 알맞은 뜻을 고릅니다.
- `별 잡기 게임`: 3개 단어 게임 중 2개를 100점 맞히면 열리는 보너스 게임입니다. 20초 동안 별을 눌러 점수를 모읍니다.

## 보상 해금 방식

보너스 게임은 아래 세 게임 중 아무 2개를 100점 맞히면 열립니다.

- `뜻 보고 단어 쓰기` 100점
- `예문 빈칸 맞추기` 100점
- `듣고 뜻 고르기` 100점

해금 상태는 브라우저 `localStorage`에 저장되므로 같은 브라우저에서는 새로고침 후에도 유지됩니다.

## 데이터 저장

앱 데이터는 브라우저 `localStorage`의 `wordMasterData_lesson16` 키에 저장됩니다.

저장되는 주요 정보:

- 단어 목록
- 예문
- 각 단어 게임 100점 달성 여부

브라우저 저장소를 지우면 앱은 기본 단어 목록과 잠긴 보너스 상태로 돌아갑니다.

## 기술 구성

- HTML, CSS, JavaScript만 사용한 단일 파일 앱
- Web Speech API로 영어 단어 발음 재생
- Web Audio API로 정답/오답 효과음 재생
- 서버, 패키지 설치, 빌드 도구 불필요
